# Udacity Data Analysis Nanodegree — practice data

Course datasets and exercise files from the Data Analyst Nanodegree (spreadsheets, metrics, forecasting, and related drills). Filenames are normalized (no duplicate downloads, no Windows ` (1)` copies).

## What is in this repo

| Area | Files |
|------|--------|
| **Business & marketing metrics** | `band-*.csv`, `band-churn-rate-quiz.xlsx`, `business-metrics-lesson-terminology-and-formulas-1.pdf` |
| **Financial statements & margins** | `smoothierock-*.csv`, `smoothierock-marginquizzes.xlsx` |
| **Forecasting** | `practice-*`, `solution-practice-quiz-salesforecast-topdown.xlsx`, `sales-forecasting-examples.xlsx.zip` |
| **Sports salaries (multiple formats)** | `albb-salaries-2003*.csv` — same general topic, different delimiters/column layouts for spreadsheet practice |
| **Geography / reference** | `worldcities.csv`, `air-routes-sfo.xlsx`, `american-colleges.xlsx`, `sp500-companies.xlsx` |
| **Excel practice** | `extract-text*.xlsx`, `fruit-*.xlsx`, `index-demo.xlsx`, `lake-superior.xlsx`, `special-orders.xlsx`, `puget-sound-butter-clams.xlsx` |

### Notes

- **`band-marketingmetric-quizzes-a.csv`** — raw campaign data. **`band-marketingmetric-quizzes-a-with-metrics.csv`** — same rows with CTR, CPC, CPL columns added.
- **`albb-salaries-2003-02.csv` … `-04.csv`** — additional downloads with different schemas (e.g. tab vs comma, combined `Player` vs `Last`/`First`); kept as separate files on purpose.
